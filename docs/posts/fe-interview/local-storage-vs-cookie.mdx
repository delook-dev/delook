## 답변

로컬 스토리지는 클라이언트에만 저장되는 데이터로 용량이 더 크고, 쿠키는 서버에도 자동 전송되어서 인증에 사용됩니다.

|                 | _로컬 스토리지_                              | _쿠키_                                     |
| :-------------: | -------------------------------------------- | ------------------------------------------ |
| **서버로 전송** | 하지 않음. 클라이언트에만 저장               | 모든 요청마다 쿠키가 함께 전송             |
| **데이터 크기** | 5MB                                          | 4KB                                        |
| **데이터 타입** | 자바스크립트 객체                            | 문자열                                     |
|    **만료**     | 수동으로 지우거나 삭제할 때까지 유지         | 특정 날짜 또는 시간에 만료되도록 설정 가능 |
|    **사용**     | 방문 시 지속되어야 하는 대량의 데이터에 적합 | 소량의 데이터 및 사용자 행동 추적에 적합   |
